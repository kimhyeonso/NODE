# 데이터 폴더

## catalog

현재 사이트에서 실제로 불러오는 상품 데이터입니다.

- `products.json`: 상품 기본 정보, 상세 이미지, 제품 사양

## legacy

기존에 작성된 이미지 목록과 메모를 보관하는 폴더입니다.
현재 페이지에서는 직접 불러오지 않습니다.

- `product-image-list.json`: 기존 PRODUCT 이미지 목록
- `accessory-image-list.json`: 기존 ACCESSORY 이미지 목록
- `lifestyle-image-notes.txt`: JSON 형식이 아니었던 LIFE STYLE 메모와 이미지 목록
