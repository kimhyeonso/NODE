# 🎵 NODE

### 소리의 연결점으로 완성되는 프리미엄 오디오 플랫폼

> **HEAR EVERY EMOTION**
>
> *Capturing Every Detail of Sound.*

NODE는 전문가와 음악 애호가를 위한 프리미엄 스피커 쇼핑 플랫폼입니다.

단순한 제품 판매를 넘어, 음악과 사운드의 감동을 시각적으로 전달하여 사용자가 전시회와 콘서트에 방문한 듯한 몰입감 있는 경험을 제공합니다.

브랜드 아이덴티티인 **"소리의 연결점(Node)"** 을 중심으로 제품 정보, 아티스트 스토리, 음원 콘텐츠, 인터랙티브 비주얼 요소를 제공하여 사용자가 제품의 가치를 직관적으로 이해할 수 있도록 설계하였습니다.

---

## 📌 Project Overview

NODE는 스튜디오 크리에이터와 라이브 퍼포먼서들이 자신의 환경에 가장 적합한 하이엔드 스피커 및 음향 장비를 탐색하고 비교하며 선택할 수 있도록 돕는 전문 온라인 플랫폼입니다.

### 🎯 Project Goals

* 사용자 중심의 UI/UX 설계 경험 습득
* 프리미엄 오디오 쇼핑몰 구현
* Canvas 기반 인터랙티브 콘텐츠 제작
* Open API 및 JSON 데이터 활용
* 실무형 쇼핑몰 기능 구현
* GitHub 협업 및 버전 관리 경험 습득
* 브랜드 아이덴티티를 반영한 웹 디자인 구현
* 몰입형 사용자 경험 제공
